💧 Calculadora de Consumo de Água

📌 Descrição

Projeto desenvolvido em Python para calcular o consumo de água com base em dados informados pelo usuário.

🚀 Funcionalidades

- Cálculo de consumo de água
- Exibição de resultados detalhados
- Interface via terminal

🛠 Tecnologias utilizadas

- Python

▶️ Como executar

python main.py

📄 Observação

Projeto criado com foco em prática de lógica de programação e aplicação em um problema real.
